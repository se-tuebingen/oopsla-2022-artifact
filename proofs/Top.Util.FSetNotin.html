Top.Util.FSetNotin

Lemmas and tactics for working with and solving goals related to non-membership in finite sets. The main tactic of interest here is notin_solve.

Facts about set (non-)membership


Lemma in_singleton : ‚àÄ x,
  In x (singleton x).

Lemma notin_empty : ‚àÄ x,
  ¬¨ In x empty.

Lemma notin_union : ‚àÄ x E F,
  ¬¨ In x E ‚Üí ¬¨ In x F ‚Üí ¬¨ In x (union E F).

Lemma elim_notin_union : ‚àÄ x E F,
  ¬¨ In x (union E F) ‚Üí (¬¨ In x E) ‚àß (¬¨ In x F).

Lemma notin_singleton : ‚àÄ x y,
  ¬¨ E.eq x y ‚Üí ¬¨ In x (singleton y).

Lemma elim_notin_singleton : ‚àÄ x y,
  ¬¨ In x (singleton y) ‚Üí ¬¨ E.eq x y.

Lemma elim_notin_singleton' : ‚àÄ x y,
  ¬¨ In x (singleton y) ‚Üí x ‚â† y.

Lemma notin_singleton_swap : ‚àÄ x y,
  ¬¨ In x (singleton y) ‚Üí ¬¨ In y (singleton x).

(* *********************************************************************** *)

Rewriting non-membership facts


Lemma notin_singleton_rw : ‚àÄ x y,
  ¬¨ In x (singleton y) ‚Üî ¬¨ E.eq x y.

(* *********************************************************************** *)

Tactics

The tactic notin_simpl_hyps destructs all hypotheses of the form (~ In x E), where E is built using only empty, union, and singleton.


The tactic notin_solve solves goals of them form (x ‚â† y) and (~ In x E) that are provable from hypotheses of the form destructed by notin_simpl_hyps.
